# EASY LINE MAN Print

พิมพ์ใบเสร็จ LINE MAN (Wongnai Merchant App) ออกเครื่องพิมพ์บลูทูธ 58 มม.
โดยไม่ต้องใช้เครื่อง Sunmi — รันบนมือถือ Android แรงๆ เครื่องไหนก็ได้

## หลักการทำงาน (โหมด A)
1. ออเดอร์เข้า WMA → เปิดหน้ารายละเอียดออเดอร์
2. แอพอ่านหน้านั้นเอง → พิมพ์ใบเสร็จออกเครื่องพิมพ์บลูทูธอัตโนมัติ
3. เปิดออเดอร์เดิมซ้ำ = ไม่พิมพ์ซ้ำ (กันด้วยรหัส LMF)
4. ถ้าอ่านไม่ครบ → พิมพ์ใบ "เปิดแอพดูออเดอร์" + เสียงเตือนยาว (ออเดอร์ไม่หาย)

แอพอ่าน **เฉพาะแอพ WMA (com.wongnai.rms)** เท่านั้น ไม่ยุ่งแอพอื่น

## วิธี build APK (ไม่ต้องลงโปรแกรมอะไรเลย)
1. สร้าง repo ใหม่บน GitHub แล้ว push โค้ดชุดนี้ขึ้นไป
2. ไปแท็บ **Actions** → กดรัน workflow "Build APK" (หรือมันรันเองตอน push)
3. รอ ~3-5 นาที → เข้าไปในรันที่เสร็จ → โหลดไฟล์ **easy-lineman-print-apk**
4. แตกไฟล์ zip จะได้ `app-debug.apk` → ส่งเข้ามือถือ → กดติดตั้ง (อนุญาต "ติดตั้งจากแหล่งไม่รู้จัก")

## วิธีตั้งค่าในมือถือ
1. จับคู่ (pair) เครื่องพิมพ์บลูทูธในตั้งค่ามือถือก่อน
2. เปิดแอพ EASY LINE MAN Print
3. กด "เปิดสิทธิ์ Accessibility" → เปิดสวิตช์ของบริการนี้
4. เลือกเครื่องพิมพ์จากรายการ → กด "บันทึกเครื่องพิมพ์"
5. กด "ทดสอบพิมพ์" — ต้องได้ใบเสร็จตัวอย่างออกมา
6. เปิด WMA ค้างไว้ พอมีออเดอร์ก็แตะเข้าไปดู → พิมพ์เอง

## ปรับแต่งทีหลัง
- เปลี่ยนหน้าตาใบเสร็จ: แก้ `ReceiptRenderer.kt`
- ถ้าอ่านรายการผิด/ไม่ครบ: แก้ `OrderParser.kt`
- จะอัปเกรดเป็นโหมด B (ไม่ต้องแตะ): เพิ่ม auto-click ใน `OrderAccessibilityService.kt`

## เปลี่ยน QR หรือข้อความท้ายใบทีหลัง
- เปลี่ยนรูป QR: เอาไฟล์ใหม่ชื่อ `line_qr.png` วางทับใน `app/src/main/res/drawable-nodpi/`
- เปลี่ยนข้อความชวนแอดไลน์: แก้ `CTA_LINE1` / `CTA_LINE2` ที่หัวไฟล์ `ReceiptRenderer.kt`
